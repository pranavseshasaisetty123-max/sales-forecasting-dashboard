# Sales Forecasting & Dashboard Project

## Project Overview
This project focuses on analyzing historical retail sales data and forecasting future weekly sales using spreadsheet-based analysis. The goal is to help businesses understand sales trends and plan inventory and operations more effectively.

## Dataset
- Walmart retail sales dataset
- Over 400,000 historical weekly sales records
- Time period: 2010–2012

## Tools Used
- Google Sheets / Microsoft Excel
- Pivot Tables
- Line Charts
- Moving Average Forecasting

## Methodology
- Cleaned and validated raw sales data
- Aggregated store- and department-level data into total weekly sales
- Analyzed sales trends using time-series line charts
- Applied a 4-week moving average to forecast the next 12 weeks of sales
- Built a dashboard with key performance indicators and trend visualization

## Key Insights
- Weekly sales remain mostly stable over time
- Occasional spikes indicate seasonal or holiday-driven demand
- Forecasted sales stay consistent around 45–46 million per week

## Dashboard
The dashboard includes:
- Average Weekly Sales
- Highest Weekly Sales
- Lowest Weekly Sales
- Weekly Sales Trend Chart

## Conclusion
This project demonstrates an end-to-end sales forecasting workflow using spreadsheets, focusing on clarity, business understanding, and actionable insights.
